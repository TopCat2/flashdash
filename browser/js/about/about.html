About Flash-Dash

Flash-Dash allows users to create as many as dashboards as they desire.
In these dashboards, a user can add various types of graphs from their desired API.
Additionaly, Flash-Dash offers pre built API URLs for Twitter/Yahoo Finance.
All the user needs to do is input their API key.
